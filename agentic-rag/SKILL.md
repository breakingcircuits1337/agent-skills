# Agentic RAG Skill

Iterative query refinement for deeper context retrieval.

## Triggers
- "Find all relevant context for [topic]"
- "Perform an iterative research search"
- "Retrieve hidden context regarding [subject]"
- "Optimize my RAG query for [objective]"

## Process
1. **Initial Search**: Perform a standard semantic search.
2. **Analysis**: Evaluate results for gaps or ambiguity.
3. **Refinement**: Rewrite the query to target missing information.
4. **Final Synthesis**: Consolidate findings into a comprehensive response.
